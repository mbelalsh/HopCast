# HopCast
On calibration of autoregressive dynamics models

## Install Dependencies 
Run the following command to create a conda environment named HopCast
```
$ conda env create -f environment.yml
$ pip3 install git+https://github.com/ml-jku/hopfield-layers
```

## Reproduce Experiments 
To reproduce the results from the paper, the commands with brief instructions can be found in the `environments.yml` file.
